## 输出要求

直接输出改编后的英文剧本，使用好莱坞标准剧本格式：

```
# Episode N: {Title}

> Keywords: {3 keywords}
> Satisfaction Points: {types}
> Previously: {recap}

---

## Scene 1

**INT./EXT. {LOCATION} - DAY/NIGHT**
**Characters:** {list}

WIDE SHOT - {Scene description}

MEDIUM SHOT - {Action description}

**{CHARACTER NAME}** ({tone}): "{dialogue}"

CU (CLOSE-UP) - {Detail description}

♪ Music: {cue}

---

> Hook: {cliffhanger}
> Next: {teaser}
```

## 适配规则

### 格式转换
- 内景 → INT. / 外景 → EXT.
- 全景 → WIDE SHOT / 中景 → MEDIUM SHOT / 近景 → CLOSE-UP / 特写 → EXTREME CLOSE-UP
- 日 → DAY / 夜 → NIGHT / 晨 → DAWN / 昏 → DUSK
- 角色名全部大写
- 首次出场时在括号内标注年龄和简要描述

### 文化适配
- 宫斗/皇室 → Royal Court / Dynasty / High Society
- 门第观念 → Class distinction / Social status
- 彩礼/嫁妆 → Prenuptial agreements / Family expectations
- 家族企业 → Family conglomerate / Dynasty corporation
- 拜师/师傅 → Mentor / Training
- 修仙/武功 → Supernatural powers / Special abilities
- 公安/警察 → Police / FBI / Detective
- 春节/中秋 → 保留文化特色但加简短解释
- 慈善晚宴替代宫廷宴会
- 律师函替代"断绝关系"

### 台词适配
- 不能直译，要用地道英文表达
- 避免中式英语（Chinglish）
- 保持台词的节奏感和冲击力
- 俚语和幽默要做本地化处理
- 保留爽感但调整文化语境

### 平台适配
- ReelShort：浪漫向为主，可适度性暗示，Billionaire/Werewolf 题材最受欢迎
- DramaBox：题材更广，古风/玄幻接受度高
- 通用：暴力尺度参考 TV-14，情感尺度参考 PG-13 到 TV-MA

### 已验证海外爆款元素
- Billionaire + Forced Marriage
- Werewolf / Alpha Mate
- Secret Baby + Reunion
- Revenge + Hidden Identity
- Mafia Romance
- Vampire / Supernatural Romance

## 重要技术决策
- LiveComponent 根元素必须稳定（PipelineModal 包裹 div 修复）
- 函数组件 assigns 必须显式传入（auto_chain/pipeline_state KeyError 修复）
- load_novel_field 用 rescue 兜底（NovelProject 可能不存在）
- art_styles.ex 用 def header 模式解决多 clause + default value warning

## 遇到的坑
1. Device Guard 阻止 erl.exe → worktree 中无法编译，需在主目录操作
2. git stash pop 3个冲突 → checkout --ours + 手动合并
3. PipelineModal `:if` 在 root div → "Stateful components must have single root" 错误
4. config_stage 缺少 auto_chain assigns → KeyError 崩溃
5. get_prompt/2 多 clause + default → Elixir 要求 def header
6. @status_messages module attribute vs assigns 同名 → 改用 @default_status_messages
